Project Overview

The AI Music Composer is a neural network-based system that generates original musical compositions across various genres. Using deep learning techniques and trained on thousands of classical and contemporary pieces, this AI creates melodies, harmonies, and rhythms that blend mathematical precision with artistic expression.

Key Features

  • Multi-genre composition generation
  • Real-time MIDI output
  • Customizable style parameters
  • Interactive web interface

Data Processing

MIDI files are converted to numerical sequences representing notes, durations, and velocities.

def midi_to_sequence(midi_file):
    notes = []
    for msg in midi_file:
        if msg.type == 'note_on':
            notes.append([msg.note, msg.velocity, msg.time])
    return np.array(notes)

Generation Process

Temperature-controlled sampling ensures musical coherence while maintaining creativity.

def generate_sequence(model, seed, length, temperature=1.0):
    sequence = seed
    for _ in range(length):
        prediction = model.predict(sequence)
        prediction = adjust_temperature(prediction, temperature)
        next_note = sample_from_distribution(prediction)
        sequence = np.append(sequence, next_note)
    return sequence
